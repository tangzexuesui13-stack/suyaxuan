# DaiP智能聊天室

一个基于WebSocket的局域网多人聊天应用，支持文字消息、表情发送、@指令等功能。

## 功能特点

- 支持多人实时聊天
- 用户昵称唯一性验证
- 在线用户列表显示
- 表情发送功能
- @指令支持：
  - @川小农：与AI助手对话
  - @电影URL：电影播放功能
- 响应式设计，支持移动端和桌面端

## 技术栈

- 服务端：Python + WebSocket
- 前端：HTML + CSS + JavaScript

## 安装和运行

### 服务端

1. 确保安装了Python 3.6+
2. 创建Python虚拟环境：
   ```
   cd d:\实训\带派\server
   python -m venv venv
   ```
3. 激活虚拟环境：
   - Windows系统：
     ```
     venv\Scripts\activate
     ```
4. 安装依赖：
   ```
   pip install websockets
   ```
5. 运行服务器：
   ```
   python server.py
   ```

### 客户端

1. 使用浏览器打开 `client/index.html`
2. 输入昵称，选择服务器，点击登录

## 配置说明

服务器配置文件位于 `server/config.json`，可以配置多个服务器地址。

### 局域网服务器设置

- 服务器默认会同时启动在 `localhost:8765` 和 `0.0.0.0:8765`
- 局域网内其他用户可以通过连接到 `ws://服务器IP:8765` 加入聊天室
- 请确保服务器的防火墙允许端口8765的访问
- 客户端登录时可以选择"局域网服务器"选项进行连接

## 使用说明

- 登录后可以在输入框发送文字消息
- 点击表情按钮可以插入表情
- 输入 `@川小农` 可以与AI助手对话
- 输入 `@电影URL` 可以请求播放电影
- 点击右上角退出按钮可以退出聊天室

## 注意事项

- 确保所有用户处于同一局域网内才能使用局域网聊天功能
- 请确保服务器的防火墙允许端口8765的访问
- 电影播放功能需要URL指向支持iframe嵌入的视频源
- AI对话功能为基础实现，后续可扩展
