DaiP智能聊天室 - 访问说明

步骤1：启动WebSocket服务器（可选）

如果您想要体验真实的多人聊天功能，需要启动服务器：

在Windows系统中，打开命令提示符（CMD）或PowerShell，按顺序运行以下命令：

cd d:\实训\带派\server
python -m venv venv
venv\Scripts\activate
pip install websockets
python server.py

这些命令会：

  • 创建Python虚拟环境
  • 激活虚拟环境
  • 安装websockets库
  • 启动WebSocket服务器

服务器启动后将同时监听：

  • 本地访问：ws://localhost:8765
  • 局域网访问：ws://服务器IP:8765

注意：如果无法启动服务器或不需要多人聊天功能，可以选择进入模拟聊天模式直接体验界面功能。

步骤2：访问聊天室

方法一：直接在浏览器中打开本地文件

d:\实训\带派\client\index.html

方法二：使用本地HTTP服务器（如果Python可用）

cd d:\实训\带派\client
python -m http.server 8000

然后在浏览器中访问：http://localhost:8000

步骤3：开始聊天

  • 输入您的昵称
  • 选择服务器地址：
    • 本地服务器：仅在本机上使用
    • 局域网服务器：用于与局域网内其他用户聊天
    • 备用服务器：备用连接选项
  • 点击登录按钮进入聊天室
  • 在聊天界面中发送消息、表情或使用@指令

模拟聊天模式

如果无法启动WebSocket服务器，系统会自动提示您进入模拟聊天模式。在此模式下：

  • 您可以看到模拟的在线用户
  • 系统会自动生成其他用户的聊天消息
  • 支持@川小农AI对话功能
  • 支持@电影URL播放功能
  • 这是一个单机体验版本，无需服务器支持

局域网聊天设置

要在局域网内进行多人聊天：

  1. 在一台电脑上启动服务器
  2. 确认该电脑的防火墙允许端口8765的访问
  3. 其他电脑在登录时选择"局域网服务器"选项
  4. 确保所有电脑连接到同一个局域网

注意事项

  • 真实多人聊天功能需要启动WebSocket服务器
  • 所有用户需要在同一局域网内才能相互通信
  • 请确保服务器的防火墙允许端口8765的访问
  • 模拟聊天模式无需服务器，适合快速体验界面功能
